문의 방법

다음과 같은 방법으로 연락하실 수 있습니다:

🐛 버그 리포트

버그를 발견하셨나요? GitHub Issues에 다음 정보와 함께 제출해주세요:

  • 버그 설명
  • 재현 단계
  • 기대되는 동작
  • 실제 동작
  • 브라우저 및 OS 정보

✨ 기능 요청

새로운 기능을 제안하고 싶으신가요? GitHub Issues에서 "[Feature Request]" 태그와 함께 다음을 포함해주세요:

  • 기능 설명
  • 사용 사례
  • 기대되는 효과

❓ 일반 질문

프로젝트 사용법이나 일반적인 질문은 GitHub Discussions를 이용해주세요.

🤝 기여하기

코드나 문서로 프로젝트에 기여하고 싶으신가요?

  1. 저장소를 Fork하세요
  2. 새 브랜치를 만드세요
  3. 변경사항을 커밋하세요
  4. Pull Request를 제출하세요
⏰ 응답 시간
