# Security Reports

보안 리포트 관리 프로젝트

## 설명

이 프로젝트는 보안 리포트를 관리하기 위한 시스템입니다.

## 시작하기

프로젝트를 로컬에서 실행하려면:

1. 저장소 클론
```bash
git clone https://github.com/bosong2/securityreports.git
cd securityreports
```

2. 프로젝트 실행
- `reports.html` 파일을 브라우저에서 열어 사용할 수 있습니다.
